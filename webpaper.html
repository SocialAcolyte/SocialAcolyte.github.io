    why this is a weapon, and why it's free
    
The point.

    
An AI that lives inside the page can rewrite anything you load — any tone, any framing, in real time. That is a gift and a loaded gun. The same machinery that deletes an ad can inject a lie. The same machinery that strips bias can manufacture it — tuned to you specifically, using everything a system knows about how you in particular can be moved.

    
We are close to a world where a resourced actor — a platform, a state, an autocrat — can run a propaganda apparatus so personal and so relentless that no individual is memetically equipped to survive contact with it. Your sense of what's real, rewritten upstream of your awareness, page by page. That actor is coming. Bad actors will inevitably build this. The only open question is whose hands hold it.

    
The one thing that destabilises a centralised memetic weapon is the same weapon in everyone's hands, working for them. If they can inject, you can extract. If they can censor, you can restore. Defense and attack are the same primitive — so the primitive must be free, auditable, and public.

    
So webpaper is open-source, so this can never be something only they have. It is strictly non-commercial, because the moment a tool like this grows a profit motive it stops working for you and starts working for its shareholders — the exact failure we're built to prevent.
