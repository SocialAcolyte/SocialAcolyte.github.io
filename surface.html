How it works

01 Connect Claude

Grant Surface access to your existing Claude account. No API key, no separate subscription — billed to your Claude Max plan.

02 Write your memory

A short document about who you are: occupation, interests, preferences. Import from your Claude Code memory files, or paste directly.

03 Browse

Surface reads each page, sends it to Claude with your memory, and marks what matters — quietly, without new chrome or popups.

Install — early build

Download surface.zip
  1. 01 Unzip the downloaded file. Keep the folder somewhere permanent — Chrome loads from it directly.
  2. 02 Open chrome://extensions and enable Developer mode (top-right toggle).
  3. 03 Click Load unpacked and select the unzipped folder.
  4. 04 Pin Surface from the extensions menu. Click it, grant claude.ai access when prompted.
  5. 05 Write a few lines about yourself in the Memory tab. Browse. Yellow marks are strong matches, blue are secondary.

Requires a Claude account (free tier works). Surface piggybacks on your existing session — it never stores your credentials.
