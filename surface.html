How it works

01 Install

Sideload the extension into Chrome. No store listing, no update prompts. Just a folder Chrome reads directly.

02 Auth

Connect your claude.ai session once. Surface harnesses Claude's intuition through dynamic DOM injection. No API key, no separate billing.

03 Calibrate

Write a short memory document about who you are, once. Then browse. Every surface of the internet starts adjusting to your context.

Install / early build

Download surface.zip
  1. 01 Unzip the downloaded file. Keep the folder somewhere permanent. Chrome loads from it directly.
  2. 02 Open chrome://extensions and enable Developer mode (top-right toggle).
  3. 03 Click Load unpacked and select the unzipped folder.
  4. 04 Pin Surface from the extensions menu. Click it, grant claude.ai access when prompted.
  5. 05 Write a few lines about yourself in the Memory tab. Browse. Yellow marks are strong matches, blue are secondary.

Requires a Claude account. Free tier works. Surface borrows your existing session and never stores your credentials.
